## Install

```bash
npm install -g @boxcrew/cli
```

Requires Node.js 20 or later. After installation, the `bx` command is available globally.

## Log in

### Browser login (recommended)

```bash
bx login
```

Opens your browser to the BoxCrew dashboard. Once you authenticate, an API key is created and saved to the CLI automatically — no copy-pasting.

| Flag | Description | Default |
|------|-------------|---------|
| `--api-url <url>` | Override the API URL | `https://api.boxcrew.ai` |
| `--frontend-url <url>` | Override the frontend URL | `https://boxcrew.ai` |

### Paste login (headless environments)

For SSH, CI/CD, or machines without a browser:

```bash
bx login --paste
```

You'll be prompted to paste an API key. Create one first in the [dashboard](https://boxcrew.ai) under **Settings > API Keys**.

### Environment variables

| Variable | Description |
|----------|-------------|
| `BOXCREW_API_URL` | Override the BoxCrew API URL |
| `BOXCREW_FRONTEND_URL` | Override the BoxCrew frontend URL |

## Log out

```bash
bx logout
```
