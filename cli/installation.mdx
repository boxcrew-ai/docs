## Install

```bash
npm install -g @boxcrew/cli
```

Requires Node.js 20 or later. After installation, the `bx` command is available globally.

## Log in

### Browser login (recommended)

```bash
bx login
```

Opens your browser to the BoxCrew dashboard where you can copy your API key. Back in the terminal, paste the key when prompted.

On headless environments (SSH, CI/CD), create an API key in the [dashboard](https://boxcrew.ai) under **Settings > API Keys** and paste it when prompted.

| Flag | Description | Default |
|------|-------------|---------|
| `--api-url <url>` | Override the API URL | `https://api.boxcrew.ai` |
| `--frontend-url <url>` | Override the frontend URL | `https://boxcrew.ai` |

### Environment variables

| Variable | Description |
|----------|-------------|
| `BOXCREW_API_URL` | Override the BoxCrew API URL |
| `BOXCREW_FRONTEND_URL` | Override the BoxCrew frontend URL |

## Log out

```bash
bx logout
```
