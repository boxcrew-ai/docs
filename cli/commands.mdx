## bx agents

### List agents

```bash
bx agents list
```

Shows all your agents with name, runtime, model, and status.

### Create an agent

```bash
bx agents create <name> [options]
```

| Option | Description | Default |
|--------|-------------|---------|
| `-r, --runtime <runtime>` | Runtime: `claude-code`, `opencode`, or `openclaw` | `claude-code` |
| `-m, --model <model>` | Model override | Runtime default |
| `-i, --instructions <text>` | System instructions for the agent | None |

Agent names must be 3–30 characters, start with a letter, and contain only letters, numbers, and hyphens.

```bash
bx agents create my-agent
bx agents create research-bot --runtime opencode --model gpt-4o
bx agents create reviewer --instructions "Review code for security issues"
```

### Chat with an agent

```bash
bx agents chat <name> <message>
```

Sends a message and streams the response. The agent can write files, run commands, and perform actions inside its VM while responding.

```bash
bx agents chat my-agent "Write tests for the auth module"
```

### Delete an agent

```bash
bx agents delete <name>
```

Permanently destroys the agent and its VM, including all files.

## bx keys

### List keys

```bash
bx keys list
```

### Create a key

```bash
bx keys create [options]
```

| Option | Description |
|--------|-------------|
| `-n, --name <name>` | Name for the key |
| `-e, --expires <date>` | Expiration date (ISO 8601) |

```bash
bx keys create --name "CI/CD"
bx keys create --name "Temp" --expires "2026-12-31T00:00:00Z"
```

<Warning>The full key (`bxk_...`) is shown only once. Store it somewhere safe.</Warning>

### Revoke a key

```bash
bx keys revoke <id>
```

## bx api

Make raw API requests. Works like `gh api`.

```bash
bx api <method> <path> [options]
```

| Option | Description |
|--------|-------------|
| `-d, --data <json>` | JSON request body |
| `-H, --header <header...>` | Additional headers (`key:value`) |

```bash
# List agents
bx api GET /agents

# Create an agent
bx api POST /agents -d '{"name":"test","runtime":"claude-code"}'

# Get billing info
bx api GET /billing

# Check install status
bx api GET /agents/my-agent/install-status

# Delete an agent
bx api DELETE /agents/my-agent
```
