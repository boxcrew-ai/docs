## Using the CLI

<Steps>
  <Step title="Install the CLI">
    ```bash
    npm install -g @boxcrew/cli
    ```
  </Step>
  <Step title="Log in">
    ```bash
    bx login
    ```
    This opens your browser. Once you log in, the CLI is authenticated automatically.
  </Step>
  <Step title="Create an agent">
    ```bash
    bx agents create my-agent
    ```
    BoxCrew provisions a sandboxed VM and installs Claude Code. This takes about 30 seconds. You can check progress with:
    ```bash
    bx api GET /agents/my-agent/install-status
    ```
  </Step>
  <Step title="Chat">
    ```bash
    bx agents chat my-agent "Create a Python CLI that fetches weather data from wttr.in"
    ```
    The agent writes code, saves files, and streams its response in real-time. Everything it creates persists on its VM filesystem.
  </Step>
</Steps>

## Using the API

<Steps>
  <Step title="Get an API key">
    Create one from the [dashboard](https://boxcrew.ai) under **Settings > API Keys**, or via the CLI:
    ```bash
    bx keys create --name "My Key"
    ```
    <Warning>The full key (`bxk_...`) is shown only once. Save it immediately.</Warning>
  </Step>
  <Step title="Create an agent">
    ```bash
    curl -X POST https://api.boxcrew.ai/agents \
      -H "Authorization: Bearer $BOXCREW_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"name": "my-agent", "runtime": "claude-code"}'
    ```
  </Step>
  <Step title="Chat (streaming)">
    ```bash
    curl -N -X POST https://api.boxcrew.ai/agents/my-agent/chat \
      -H "Authorization: Bearer $BOXCREW_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"message": "Create a Python CLI that fetches weather data from wttr.in"}'
    ```

    The response is an [SSE](https://developer.mozilla.org/en-US/docs/Web/API/Server-sent_events) stream:

    ```
    data: {"type":"response.created","response":{...}}
    data: {"type":"response.output_text.delta","delta":"I'll create"}
    data: {"type":"response.output_text.delta","delta":" a Python CLI..."}
    data: {"type":"response.completed","response":{...}}
    ```
  </Step>
</Steps>
