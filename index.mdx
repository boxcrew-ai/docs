BoxCrew gives you AI agents that can actually do things. Each agent runs Claude Code (or OpenCode, or OpenClaw) inside its own sandboxed VM with a persistent filesystem. Agents can write and run code, send emails, browse the web, and manage files — all through a simple API.

```bash
# Install the CLI
npm install -g @boxcrew/cli

# Create an agent and start chatting
bx login
bx agents create my-agent
bx agents chat my-agent "Set up a Node.js project with Express and write a REST API for a todo app"
```

## What can you build?

- **Autonomous coding agents** — agents that write, test, and iterate on code inside their own VM
- **Email-driven workflows** — each agent has its own email address; send it tasks and get results back
- **Scheduled automations** — set up cron jobs to run agents on a schedule (daily reports, monitoring, data pipelines)
- **Webhook-triggered pipelines** — connect GitHub, Slack, or any service to trigger agent work

## How it works

Every agent is an isolated [Fly Sprite](https://fly.io/docs/sprites/) VM. When you create an agent, BoxCrew provisions a VM, installs the AI runtime, and gives you an endpoint to chat with it. The agent's filesystem persists across conversations — it remembers context, keeps files, and builds on previous work.

```
You (CLI / API / Dashboard)
  → BoxCrew API
    → Fly Sprite VM (sandboxed)
      → Claude Code / OpenCode / OpenClaw
        → streams response back via SSE
```
